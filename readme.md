# Компьютерная графика. Лабораторная №3
## Теория
Метод создания реалистичного изображения, при помощи применения модели освещенности Гуро. Метод Гуро отличается от метода "простой закраски граней", тем что проводит линейную интероляцию интенсивности освещенности между вершинами граней.
Алгоритм Гуро заключается в такой последовательности:
1. Вычисляем нормали в каждой вершине треугольников, суммируя все нормали граней в которую входит данная вершина
2. Вычисляем интенсивность освещенности учитывая фоновое и диффузное освещение
3. Проводим линейную интерполяцию освещенности, проходя каждый треугольник отрезком
4. Отрисовываем отрезок с учетом проведенной интерполяции

## Структура проекта
1. assets - дополнительные файлы проекта
2. scripts - директория исходного кода JS
3. style - директория стилизующая отображение страницы 

## Запуск
Программа написана на JavaScript и работает напрямую в браузере, достаточно открыть "index.html" файл
